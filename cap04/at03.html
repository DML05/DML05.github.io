Seção 3 — Glossário Técnico

HTML
Linguagem de marcação usada na estruturação de páginas web.
CSS
Linguagem utilizada para estilização de páginas.
JavaScript
Linguagem de programação da web.
DOM
Estrutura que representa os elementos da página.
API
Interface utilizada para comunicação entre sistemas.
Pode permitir integração com bancos de dados e serviços externos.
HTTP
Protocolo utilizado na comunicação da web.
URL
Endereço utilizado para acessar páginas e recursos online.
DNS
Sistema responsável por traduzir nomes de domínio em IPs.
